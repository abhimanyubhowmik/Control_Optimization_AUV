Initialization¶

Initialization of the bus and the vehicle state for the simulation.

CreateSimulinkBus¶

Creation of the bus for simulink project

Usage

CreateSimulinkBus;

Returns

  • all necessary bus for the simulation

Warning

has to be a script because the bus are created in the base workspace

SetInitialState()¶

Set the initial vehicle status settings in the simulator

Usage

[initial_state,initial_actuators] = SetInitialState()

Returns

  • initial_state - initial vehicle status settings in the simulator

  • initial_actuators - initial actuators status settings in the simulator
